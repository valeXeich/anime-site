# Anime site

Stack
1. Django
2. Postgresql
3. Django allauth

# Launch
1. Install requirements

   ```bash
     $ pip install -r requirements.txt
   ```
2. Migrations
  
   ```bash
     $ py manage.py makemigrations
   ```
   ```bash
     $ py manage.py migrate
   ```
3. Run server

   ```bash
      $ py manage.py runserver
    ```
